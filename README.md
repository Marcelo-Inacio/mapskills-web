# mapskills-web
Projeto MapSkills desenvolvido com BOOTSTRAP, ANGULARJS, NODE, BOWER, GULP, JADE e STYLUS.

=========== Necessário ter instalado ============
  - Node [ https://nodejs.org/en/ ]
  - Bower [ $ npm install -g bower ]
  - Gulp [ $ npm install gulp-cli -g
           $ npm install gulp ]

============== Rodar localmente ==============
  
$ gulp serve

============== Gerar distribuição ==============

$ gulp ou $ gulp build
